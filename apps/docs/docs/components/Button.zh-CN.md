## API

| 属性名   | 描述               | 类型                     | 默认值      |
| -------- | ------------------ | ------------------------ | ----------- |
| type     | 按钮类型           | `string`                 | `undefined` |
| size     | 大小               | `string`: ` sm``md``lg ` | `md`        |
| disabled | 按钮是否为禁用状态 | `boolean`                | `false`     |
| plain    | 是否为朴素按钮     | `boolean`                | `false`     |
| round    | 是否为圆角按钮     | `boolean`                | `false`     |
| loading  | 是否加载中         | `boolean`                | `false`     |
| icon     | 图标               | `string`                 | `undefined` |
